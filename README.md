# Русский перевод для мода Ortalab (Balatro)

Русификатор текста для мода Ortalab в игре Balatro.

## Установка
1. Скачайте файл `rus.lua` и поменяйте название на `default.lua`
2. Скопируйте его в папку: `%AppData%/Roaming/Balatro/Mods/Ortalab/localization/`
3. Замените оригинальный файл (предварительно сделайте резервную копию)
4. Запустите игру

## Совместимость
- Balatro: версия 1.0.1o-FULL
- Ortalab: версия 1.0.1c
